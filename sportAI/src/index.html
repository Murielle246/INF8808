Analyse des rendements mensuel de buts

Cette visualisation représente les performance d’un joueur en terme d’efficience de but marqué vis-à-vis du nombre de tirs cadrés qu’il a eu a tiré.

Pour faciliter l’analyse nous avons regrouper les performances du joueur de façon mensuelle.

Indépendamment du nombre de matchs joué au cours d’un mois précis, nous avons juste fait une sommes de ses tirs cadrés durant une période d’un mois et ceci pour chaque mois de l’année, ainsi que les but marqués au cours de la même période pour tous les mois de l’année.

Performance des joueurs de soccer dans différents aspects

Cette visualisation est une carte de chaleur interactive et elle est un excellent moyen de voir vers quels aspects les différents joueurs que nous comparons penchent le plus.

La visualisation compare le nombre de buts marqués, le pourcentage de dribble réussi, le nombre de passe décisives (assistances) le nombre d’interception et le nombre de tacles réussi.

Le code de couleurs nous permet de voir quel joueur performe le plus et dans quels aspects.
